# Digital Touch

Digital Touch is a technology company focused on building practical software and IoT solutions for businesses and communities.

We specialize in developing modern systems that combine software engineering, automation, and artificial intelligence to solve real-world problems.

## What We Build

- Web applications and dashboards  
- IoT monitoring systems  
- Business automation tools  
- AI-powered solutions  
- Custom backend systems

## Key Projects

### Borehole IoT Monitoring
A smart monitoring system that tracks borehole pump activity and temperature in real time.  
The system includes IoT sensors, embedded firmware, and a web dashboard for monitoring analytics and system alerts.

### Fisc
A fiscalization and business transaction system designed to help businesses manage receipts and financial data.

### TenantSpace
A property management system that helps landlords and tenants manage rental agreements, payments, and maintenance requests efficiently.

### QuickNote
A business assistance tool that allows users to **log transactions, generate receipts, and access reports**.  
It helps small businesses and entrepreneurs track daily operations efficiently and make informed decisions.
